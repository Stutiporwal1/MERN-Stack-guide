The main purpose of this repository is to:

- Show **consistent growth** as a developer

## What is MERN Stack?

MERN Stack consists of:

- **MongoDB** – Database
- **Express.js** – Backend framework
- **React.js** – Frontend library
- **Node.js** – Runtime environment
